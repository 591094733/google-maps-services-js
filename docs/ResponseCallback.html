Interface: ResponseCallback

ResponseCallback

A callback function, which is called asynchronously when an API method completes. The callback is given either:
  • a Response object; OR
  • an error object from the underlying http library.
For example: 
  googleMapsClient.geocode({...}, function(err, response) {
    if (err) {
      // Handle error.
    } else {
      // Handle response.
    }
  });

API methods don't require a callback function, if you use the Promise API. See RequestHandle#asPromise().

Note that API methods may also throw errors synchronously, if the query was malformed. This usually indicates a programming mistake.
