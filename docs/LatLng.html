Interface: LatLng

LatLng

A latitude, longitude pair. The API methods accept either:
  • a two-item array of [latitude, longitude];
  • a comma-separated string;
  • an object with 'lat', 'lng' properties; or
  • an object with 'latitude', 'longitude' properties.
